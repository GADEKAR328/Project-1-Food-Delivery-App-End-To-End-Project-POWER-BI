# Project-1-Food-Delivery-App-End-To-End-Project-POWER-BI
This repository showcases an end-to-end Power BI dashboard project based on a food delivery dataset (Zomato-inspired). The project focuses on analyzing sales, customer behavior, and operational performance to generate actionable insights.

# Comprehensive overview:
- Total Orders & Quantity Trends (15.1K+ items across 2.7K transactions)
- Monthly Performance Analysis to track demand fluctuations
- Food Category Insights (Main Course, Snacks, Desserts, etc.) with member segmentation (Gold vs Regular)
- Top-Selling Items like Samosa, Butter Chicken, and Biryani
- Delivery Performance showing ~94% successful deliveries
- Payment Method Analysis highlighting UPI as the dominant mode
- Time-based Trends to monitor growth and seasonal patterns

# 🔧 Tools & Skills Used:
Power BI, Data Cleaning, Data Modeling, DAX, Data Visualization, Dashboard Design

# 🎯 Key Outcome:
Built an interactive and user-friendly dashboard that helps stakeholders make data-driven decisions by identifying top-performing products, customer preferences, and operational trends.
